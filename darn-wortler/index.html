Darn Wortler

Inspired by the German word game "Gefüllte Kalbsbrust"

The Objective
⏱️

Score as many points as possible before the 5-minute timer expires.

How to Play
🌱

Top Row: A random five-letter seed word for this puzzle round.

🥪

Game Grid: Guess five-letter words that fit between the given outer letters.

📊

Row Difficulty: Tougher rows yield higher scores.

Bonus Points
✨

Great Vocabulary: Earn +50 points for each obscure word.

🔥

Speed Streak: Type 3 valid words within 15s to unlock a +5 point bonus.
